## Prime Checker

```python
check = 0
def prime_checker(number):
    global check
    for i in range(1,number+1):
        modulo = number%i
        if modulo == 0:
            check += 1
    if check == 2:
        print("It's a prime number.")
    else:
        print("It's not a prime number.")
```
